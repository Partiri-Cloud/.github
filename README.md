  ## What we do

  Partiri Cloud is a platform for shipping apps without the operational overhead.
  Bring a Git repository or a container image, and go from commit to a running — no infrastructure to
  babysit.

  Learn more at [partiri.cloud](https://partiri.cloud).

  ## Open source

  We build in the open. Here's what you can install, fork, and contribute to today:

  **[harbor-cli](https://github.com/Partiri-Cloud/harbor-cli)** &nbsp;·&nbsp;
  ![Rust](https://img.shields.io/badge/Rust-000000?logo=rust&logoColor=white)
  The official `partiri` command-line interface — deploy and manage your services
  straight from the terminal. Install with `cargo install partiri-cli` or `npm install -g
  @partiri/cli`.

  **[harbor-mcp](https://github.com/Partiri-Cloud/harbor-mcp)** &nbsp;·&nbsp; ![TypeScrip
  t](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)
  A [Model Context Protocol](https://modelcontextprotocol.io) server that lets AI agents
  — Claude Code, VS Code, and others — manage your deployments, services, and metrics on
  your behalf.

  **[message-in-a-bottle](https://github.com/Partiri-Cloud/message-in-a-bottle)**
  &nbsp;·&nbsp; ![Go](https://img.shields.io/badge/Go-00ADD8?logo=go&logoColor=white)
  A multi-channel notification platform with workflow orchestration, subscriber
  management, and real-time delivery across email, SMS, push, chat, and in-app.
